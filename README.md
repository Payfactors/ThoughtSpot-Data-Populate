# ThoughtSpot Data Populate

This repository contains code to generate sample data for ThoughtSpot-related tables, including:

- **Employee Model**: Generates employee records with relevant attributes.
- **Job Model**: Populates job-related data such as titles, departments, and roles.
- **Structure Model**: Creates organizational structure data, such as reporting lines and hierarchies.
- **Org Data Model**: Produces organization-wide flattened data which included all employees ids, jobs ids, Structures ids & pricings ids.

Make sure to populate the env file in src folder before you execute the process
